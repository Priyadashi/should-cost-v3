# Supplier Data Request Form Template

## Purpose
Generate a one-page structured data request for the buyer to send to the supplier. This collects the specific data points needed to validate and refine the should-cost model, closing the loop between modeled assumptions and actual operations.

## Template

---

### Cost Transparency Data Request — [Part Name/Number]

**To:** [Supplier Name]
**From:** [Buyer Name / Company]
**Date:** [Date]
**Re:** Cost model data validation for [Part Name], [Annual Volume] units/year

To ensure our cost model reflects your actual operations accurately, please provide the following information. This data will be used to update our assumptions and arrive at a fair, fact-based price. All information will be treated as confidential.

## Usage Notes

When generating this form, tailor it to the specific analysis:
- Remove sections that are not applicable (e.g., surface treatment if no finish is specified)
- Add rows to the manufacturing process table if more operations exist
- Pre-fill the "Our Assumption" column with the model's values where available, so the supplier can see and respond to specific assumptions
- Adjust the response timeframe based on relationship urgency
