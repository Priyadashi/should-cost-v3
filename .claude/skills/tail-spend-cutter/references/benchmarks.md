## Pareto Analysis Benchmarks

### Typical Spend Distribution

| Segment | % of Suppliers | % of Spend | Typical Label |
|---------|---------------|------------|---------------|
| Head | 5-10% | 50-60% | Strategic |
| Core | 10-20% | 20-30% | Managed |
| Tail | 70-85% | 10-25% | Tail |

### Tail Spend Characteristics

| Metric | Typical Range | Red Flag |
|--------|--------------|----------|
| Avg spend per tail supplier | $500-$15,000 | >$25,000 (misclassified) |
| Avg transactions per tail supplier | 1-5 per year | >12 (high frequency = opportunity) |
| % of tail with no contract | 60-80% | >90% (no governance) |
| % of tail with single transaction | 30-50% | >60% (one-time buying) |
| Categories with >10 tail suppliers | 3-8 categories | >15 (fragmentation) |

## Consulting Firm Reference Rates

### Typical Tail Spend Program Results (McKinsey/BCG/Kearney benchmarks)

| Metric | Conservative | Moderate | Aggressive |
|--------|-------------|----------|-----------|
| Overall tail savings | 5-8% | 8-12% | 12-18% |
| Supplier count reduction | 15-25% | 25-40% | 40-60% |
| Quick-win savings (0-90 days) | 2-3% | 3-5% | 5-8% |
| Category consolidation savings | 8-15% | 15-22% | 22-30% |
| Process cost reduction | 10-20% | 20-35% | 35-50% |
| Time to full realization | 12-18 months | 9-12 months | 6-9 months |

### Feasibility Factor Guidelines

| Factor | Value | When to Use |
|--------|-------|-------------|
| 0.3 | Conservative | First-time analysis, poor data quality, no dedicated procurement |
| 0.4 | Below average | Some procurement function, limited spend visibility |
| 0.5 | Moderate | Established procurement, moderate data quality |
| 0.6 | Above average | Mature procurement, good data, some management support |
| 0.7 | Aggressive | Strong procurement team, clean data, executive support |
| 0.8 | Best case | World-class procurement, full organizational alignment |
